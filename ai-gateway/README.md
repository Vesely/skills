# ai-gateway

Generate text and images from the CLI via the Vercel AI Gateway. One key, hundreds of models — including Nano Banana, Flux, Imagen, Claude, GPT, and Grok.

Wraps the [`@vesely/ai-gateway-cli`](https://github.com/Vesely/ai-gateway-cli) tool so any Claude Code agent can generate content with a single command.

## Install

```
npx skills@latest add Vesely/skills/ai-gateway
```

## Requires

- Node 20+
- An AI Gateway API key from <https://vercel.com/ai-gateway>
- The CLI itself: `npm install -g @vesely/ai-gateway-cli`
