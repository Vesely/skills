# cursor-agent

Delegate a task to Cursor's headless CLI agent (`cursor-agent`) for a second opinion from a non-Claude model. Great for code reviews, plan critiques, and verifying work with a different model.

## Install

```
npx skills@latest add Vesely/skills/cursor-agent
```

## Requires

- [`cursor-agent`](https://docs.cursor.com/en/cli/overview) installed and authenticated on your machine.
